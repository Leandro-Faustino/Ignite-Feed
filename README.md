# Ignite Feed
Este é um projeto feito para aprendizado de alguns fundamentos do React.js.

## Conceitos aprendidos
1. Componentização
2. propriedades
3. Estado
4. imutabilidade
5. styles-components

## Para rodar:
1. Para Instalar as dependências, vamos rodar ->  npm install
2. Para executar o projeto, vamos rodar ->  npm run dev
